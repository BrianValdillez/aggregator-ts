# Aggre-Gator

## Intro
Aggregate your RSS feeds with gator since Google couldn't be arsed to keep their service up and running!

## Commands
This is a command line tool. Here are the commands.

| Command | Description | Param #1 | Param #2 |
| -- | -- | --- | -- |
| `login` |  Log in to any registered user. | The username to log in with | |
| `register` | Register a new user. | New user's username | |
| `reset` | Reset the entire database. | | |
| `users` | Display a list of users. | | |
| `agg` | Aggregate all registered feeds. Use Ctrl+X to kill process. | | |
| `feeds` | Displays a list of all registered feeds. | | |
| `addfeed` | Add and follow a new feed | Feed Name | Feed URL |
| `follow` | Follow an existing feed. | The Feed URL |
| `unfollow` | Unfollow an existing feed. | Thee Feed URL |
| `following` | Display a list of feeds followed by the current user. |
| `browse` | Display the most recent posts from the user's followed feeds. | The number of posts to display (Default: 2) |
